# Digital-Clock
Digital-Clock based on Zedboard(ZYNQ) vivado 2017.4

including: Showing time, change time, stopwatch, timer, alarm and bugs :P

Peripherals used: LEDs, Push button, OLED SSD1306(SPI), Passive buzzer
